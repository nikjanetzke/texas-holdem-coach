# Texas Hold'em Coach

An interactive poker coaching game for learning Texas Hold'em.

## Goal

Build a beginner-friendly training app that teaches people how to play Texas Hold'em by letting them practise hands, make decisions, and get clear feedback.

## Early product ideas

- Guided tutorials for hand rankings, blinds, betting rounds, and table flow
- Practice mode with realistic hands and simple explanations
- Coach feedback after each decision: fold, call, raise, check, bet
- Beginner-friendly strategy prompts: position, pot odds, board texture, hand strength
- Scenario drills: pre-flop, flop, turn, river
- Mistake tracking and progress history
- Optional AI-style coach explanations later

## Initial scope

Start as a browser-based app, likely static-first:

- HTML/CSS/JS or React/Vite
- Single-player training mode first
- No real-money gambling, no payments, no user accounts at the beginning
- Educational coaching focus only
